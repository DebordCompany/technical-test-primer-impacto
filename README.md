# Prueba técnica Primer Impacto

## Levantar servicios

- Cambiar en el `.env` la variable `VITE_BACK_API` por la IP de donde se esté
  levantando
- Si fuera necesario porque estén ocupados los puertos, cambiar los puertos
- En la carpeta raíz, ejecutar `docker compose up -d`
- Una vez levantados todos los servicios, estarán expuestos en sus respectivos
  puertos
